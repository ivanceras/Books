## Brackets

Svgbob can create expandable brackets

```bob

    .-                     -.
    |                       |
    |                       |
    )                       (
   <   open           close  >
    )  bracket      bracket (
    |                       |
    |                       |
    '-                     -'


   

     .-            -.
     )              (
    <    Medium      >
     )    Brackets  (
     '-            -'

    .-                     -.
    |                       |
    |                       |
    |                       |
    |  open           close |
    |  braces        braces |
    |                       |
    |                       |
    '-                     -'

    +-                     -+
    |      sharp braces     |
    |                       |
    +-                     -+
```
